# SPEC.md

## 1. 目的

『返し鈴』は、現代の一晩に焦点を当てた5分程度の和風ホラー短編サウンドノベルである。

## 2. 範囲

v0.1では以下を実装対象とする。

- タイトル画面相当の導入
- 本文表示
- 文書表示
- 選択肢3箇所
- スコア分岐
- BAD / NORMAL / TRUE の3エンド
- バックログ
- 保存 / 読込
- Web Audio による仮音
- CSS による仮背景

## 3. 非対象

- 外部音源・画像素材の同梱
- ボイス
- CG鑑賞
- 既読スキップ
- オート再生
- 多言語対応
- ストア公開

## 4. シナリオ表記ルール

- 声は「」で括る。
- 覚書・走り書き・札・写真裏・絵馬文字など文書内容は『』で括る。
- 句読点なしは返鈴覚書のみ。
- 祖母の走り書きは現代語で句点のみ。読点は使わない。
- 本文地の文は句読点を自然に使う。

## 5. エンディング判定

- 分岐3で「返事をする」場合は BAD END。
- それ以外は score >= 2 で TRUE、score >= 0 で NORMAL、score < 0 で BAD。
