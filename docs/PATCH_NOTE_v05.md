# PATCH_NOTE_v05.md

## 変更内容

ページ送り時に本文の開始位置が動かないよう、本文ウィンドウの高さを固定しました。

## 修正内容

```css
.text-window {
  height: 36vh;
  min-height: 36vh;
  max-height: 36vh;
}
```

## 意図

文章量が少ないページと多いページで、本文ウィンドウの高さが変わると、最初の行の位置が上下します。  
視線移動が増えて読みづらくなるため、本文ウィンドウは固定高さにしました。
