# SPEED_SLIDER_POLICY_v02d.md

## 正本

表示速度はスライダーで調整する。

```text
0 とてもゆっくり
1 ゆっくり
2 標準
3 速い
4 瞬時
```

## デフォルト

```text
2 標準
```

## 方針

旧UIで最もゆっくり側に見えていた設定を、新しい標準相当に寄せた。  
全体の文字送りは、従来よりゆっくり側へ寄せる。

## ms/char

```text
本文:
0=85 / 1=70 / 2=55 / 3=35 / 4=0

台詞:
0=105 / 1=86 / 2=68 / 3=45 / 4=0

文書:
0=42 / 1=34 / 2=26 / 3=16 / 4=0
```

## 旧localStorage値の扱い

```text
slow -> 2
normal -> 3
fast -> 3
instant -> 4
```
