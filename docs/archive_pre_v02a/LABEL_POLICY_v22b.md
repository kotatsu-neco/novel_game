# LABEL_POLICY_v22b.md

## 表示設定UIの表示名

内部キーは互換性維持のため変更しない。

```text
text     → 本文
voice    → 台詞
document → 文書
```

## 設定画面レイアウト

```text
表示速度        文字サイズ
本文            本文
台詞            台詞
文書            文書
```

## 理由

- 「作中本文／作中台詞／作中文書」は意味は明確だが、設定画面では長く視認性が落ちる。
- カタカナ表記はフォント種別の serif と混同し得るため避け、「台詞」を使う。
- 内部キー `text / voice / document` は既存シナリオ互換のため維持する。
