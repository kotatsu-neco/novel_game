# WORDING_REVISION_REPORT.md

## 1. 目的

『迷路』のシナリオ本文・選択肢で、ピクトグラム説明として使っていた不適切かつ過剰な語を削り、ヒントが露骨に読めてしまう箇所を抑制した。

## 2. 修正方針

```text
- ピクトグラムを「正解マーク」として読ませない。
- 不自然に説明しない。
- 動画字幕を攻略メモにしすぎない。
- 選択肢の文言で正解が分かりすぎないようにする。
- 主人公がその場で見たもの・感じたものに寄せる。
```

## 3. 主な修正

```text
- 繰り返し出ていた身体性の強い表現を全廃。
- 「まっすぐ立つ人影」「輪郭が途切れていない」「人の形を保っている」等へ置換。
- 動画字幕のヒントを直接命令ではなく、曖昧な観察メモへ変更。
- 最終階段の正解選択肢から、答えをそのまま示す言い回しを削除。
```

## 4. 注意

この修正は、文体パターンの変更ではなく、本文設計・情報開示量・選択肢設計の修正である。
