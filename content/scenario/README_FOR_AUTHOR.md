## 3. 重要ルール

- スマホそのものが原因ではない
- 俯いて地下へ降りることが儀式になっている
- 振り返ってしまうことがトリガー
- 文字は読めない
- ピクトグラムは意味だけ分かる
- 欠けた人型の道は危険
- 水泳ピクトグラムの先には水の異変がある
- 間違った道を選ぶとループする
- 3回目のループでBAD END
- 迷路では空腹も喉の渇きもない
- TRUE ENDではオフライン保存動画のヒントが鍵になる
