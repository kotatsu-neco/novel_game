# 返し鈴 v0.1

和風ホラー短編サウンドノベルの最小プレイ版です。

## 起動方法

`index.html` をブラウザで開きます。

ローカルで `fetch()` が制限されるブラウザでは、簡易サーバーで起動してください。

```bash
python -m http.server 8000
```

その後、ブラウザで以下を開きます。

```text
http://127.0.0.1:8000/
```

## 操作

- 本文ウィンドウをクリック / タップ：次へ進む
- 選択肢：タップして選択
- メニュー：バックログ表示・最初から
- 保存：現在地点を保存
- 読込：保存地点を読込
- 音：Web Audio の仮音をオン / オフ

## 重要

v0.1 では第三者素材を同梱していません。背景は CSS 表現、音は Web Audio による仮音です。
販売用素材を入れる場合は `docs/LICENSES.md` と `docs/AUDIO_SOURCES.md` を必ず更新してください。
